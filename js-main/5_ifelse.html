Conditional Statements

 

        조건문
        -if: 지정된 조건이 참인 경우, 실행할 코드브록을 지정하는데 사용
        syntax
        if(condition){}
        -else: 동일한 조건이 거짓일 경우, 실행할 코드블록을 지정하는데 사용
        -elseif: 첫번째 조건이 거짓인 경우 실행할 코드블록을 지정하는데 사용
        -switch: 실행할 여려 대체 코드블록을 지정하는데 사용

        swith : 조건이 여러개일 경우 깔끔한 코딩을 위해 사용
        swith(expression) {   //expression:표현식
            case x:
             코드블록
             break;
        }
